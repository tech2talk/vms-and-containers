1.      Download Ubuntu 18.04.5 64 bit desktop image from here.

https://releases.ubuntu.com/18.04/ubuntu-18.04.5-desktop-amd64.iso

 

2.      Start virtual box from start menu.

 

3.      Create a new VM from "Machine" à New.

 

 

4.      Select the following and then click Create.

 

Name: ubuntu workstation 18.04

Machine Folder: C:\work\VirtualBox VMs

Memory size: 4096

 

 

 

5.      Next we will add Virtual Disk to this new machine.

 

Use the following settings and then

 

File Size: 70 GB

Hard disk file type: VMDK (virtual machine disk)

Storage on physical hard disk: Fixed size.

 

Click Create after setting this configs. It will take some time to create a 70 GB file on the disk.

              

 

6.      You should see a new VM. Click on it and then click settings.

 

 

7.      Under "General" à "Advanced" select Bidirectional if it not already selected.

 

8.      Under "System" à Motherboard, unselect floppy

 

 

 

9.      Under "System" à "Processor", increase the value to 2 and selected "Enable PAE/NX"

Note: you may not have this option as it depends on the type of actual CPU of your laptop.

 

 

10.   Under "System" à "Acceleration" check the "Enable Nested Paging" if it is not enabled.

Note: you may not have this option as it depends on the type of actual CPU of your laptop.

 

11.   Under "Display" à "Screen" changed video memory to 64 MB.

 

 

12.   Under "Storage" à "Controller: IDE", click first + sign.

 

 

13.   Click Add

 

14.   Select "ubuntu-18.04.5-desktop-amd64.iso" and then click open.

 

15.   Select the newly created disk and click choose

 

16.   Click on the Empty disk and then delete it.

 

17.   Your final storage config should look like following screen.

 

18.   Under "User Interface", select "Show at Top of Screen" and then click OK at the bottom.

 

 

 

19.   At this point, close or exit out of VritualBox completely.

 

20.   Next, click "Start" menu button of your Windows and type "cmd"

Right click on "Command Prompt" and click "Run as administrator"

 

 

21.   Click "Yes" to the "user control" message.

 

22.   Copy and paste the following command to command prompt and hit enter.

This will enable the hardware virtualization support if available.

 

"C:\Program Files\Oracle\VirtualBox\VBoxManage" modifyvm "ubuntu workstation 18.04" --nested-hw-virt on

 

 

23.   Start VirtualBox from Start menu.

 

24.   Select "Ubuntu workstation 18.04" and click settings.

 

 

25.   Go under "System" à "Processor" and check if "Enable Nested VT-x/AMC-V" is enabled.

Note: If it is still disabled then you processor does not support this feature so you can ignore it.

 

 

26.   Now you virtual machine is ready. You should see summary similar to the following screen print.

 

27.   Next go to Step3 document.
